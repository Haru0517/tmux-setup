# dotfiles

![image](https://user-images.githubusercontent.com/12545287/91533865-916ed400-e94b-11ea-866e-13c94ca4062d.png)

This contains following settings:
- git
- zsh
- vim
- tmux
- macos terminal

Main color theme is Tomorrow Night Eighties.

## Installation
```shellscript
 sh setup.sh
```
